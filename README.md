This application is a task manager.
This project implements four related entities: User, Label, Task, and TaskStatus. The project is written in SpringBoot and implements standard REST API principles. The application supports authentication and authorization. SonarQube, Springdoc, and Sentry are integrated.

Application link: https://java-project-99-oh2h.onrender.com

## Start

```bash
make
```

## Setup

```bash
make build
```

## Run

```bash
make run
```

## Run tests

```bash
make test
```

## Run checkstyle

```bash
make lint
```

## Check update dependencies and plugins

```bash
make update
```
